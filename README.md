# 确定性密码生成器 (Deterministic Vault)

你是否有这样一个烦恼：
注册的网站和应用越来越多，密码设得简单怕被盗，设得复杂又总是记不住？为了图省事，所有账号都用同一个密码，结果一个网站数据泄露，所有账号跟着遭殃；想用传统的密码管理器，又担心把所有密码打包存在云端，一旦服务商被黑客攻击，后果不堪设想。
如果你也有这样的安全焦虑，那么这款 确定性密码生成器 (Deterministic Vault) 正是你需要的终极解决方案！
💡 核心理念：不“存储”密码，而是“计算”密码
它打破了传统密码管理器的思路。在这里，你不需要把密码存在任何地方。
你只需要记住一个只有你自己知道的“主密钥”，结合你要登录的“账号标识”（比如：微信、淘宝、Google），系统就能通过高强度的加密算法（HMAC-SHA256），为你瞬间“算”出一个独一无二的复杂密码。
只要“主密钥”和“账号”不变，每次算出来的密码都完全一样。把记忆的负担交给算法，把安全的控制权交还给自己。
🌟 为什么选择 Deterministic Vault？
🛡️ 完全离线使用，绝对的隐私安全
本应用没有任何后端服务器，所有的密码计算都在你的本地设备（浏览器）中离线完成。我们不存储、不上传你的主密钥和生成的密码。即使在断网环境下，你依然可以随时随地生成密码。你的秘密，永远只留在你的设备里。
📖 代码完全开源，安全经得起检验
对于一款安全工具来说，最大的信任来源于透明。本项目已在 GitHub 完全开源。任何人都可以审查代码，确保没有任何后门或恶意的数据收集行为。你完全可以放心地使用，甚至自己打包成独立的 App。
⚡ 无状态保险箱，用完即走
主密钥绝不会被保存在任何缓存中。即使别人拿到了你的手机或电脑，没有你大脑中的那个“主密钥”，他们看到的只是一堆毫无意义的账号名称，无法逆向破解出任何真实密码。
📦 便捷的本地管理与备份
虽然不存密码，但为了方便日常使用，应用支持将“账号标识”保存在本地浏览器中，并支持自定义精美图标。同时提供 CSV 导入/导出功能，让你轻松实现跨设备的数据备份与迁移。

## 🌟 核心理念

**“只需记住一个主密钥，即可遗忘所有网站的复杂密码。”**

本应用不将您的主密钥或生成的密码上传到任何云端服务器。所有的密码计算都在您的本地浏览器中离线完成。只要您输入相同的“主密钥”和“账户标识”，系统每次都会通过加密算法（HMAC-SHA256）为您生成完全相同的高强度密码。

## ✨ 主要功能

- 🔒 **完全离线计算**：无后端服务器，无云端存储，拒绝数据泄露风险。
- 🔑 **确定性生成算法**：基于 HMAC-SHA256，保证相同输入必定得到相同输出。
- 📱 **本地密码库**：可将账号标识安全保存在本地浏览器缓存中（不保存主密钥），方便随时查看和生成。
- 🎨 **自定义图标**：支持为不同的账号设置专属的文字图标和颜色。
- 📁 **数据导入导出**：支持将本地密码库导出为 CSV 文件进行备份，或从 CSV 文件导入记录。
- 🛡️ **防暴力破解机制**：密码库解锁失败 5 次将自动销毁本地库数据，保护隐私。

## 🚀 快速开始 (Web 端开发)

### 安装依赖
```bash
npm install
```

### 本地运行
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

## 📦 打包为 Android APK

如果您希望将此 Web 应用打包为 Android APK，推荐使用 **Capacitor** 或 **Cordova**。以下是使用 Capacitor 的简要步骤：

1. **安装 Capacitor 依赖**：
   ```bash
   npm install @capacitor/core @capacitor/android
   npm install -D @capacitor/cli
   ```
2. **初始化 Capacitor**：
   ```bash
   npx cap init "Deterministic Vault" "com.yourname.vault" --web-dir dist
   ```
3. **构建 Web 项目**：
   ```bash
   npm run build
   ```
4. **添加 Android 平台并同步代码**：
   ```bash
   npx cap add android
   npx cap sync android
   ```
5. **在 Android Studio 中打开并打包**：
   ```bash
   npx cap open android
   ```
   在 Android Studio 中，您可以生成签名 APK (Build -> Generate Signed Bundle / APK)。

## ⚠️ 安全免责声明

本工具旨在提供一种便捷且相对安全的密码管理方式。虽然计算过程完全在本地进行，但请务必：
1. **妥善保管您的主密钥**：一旦遗忘，您将无法找回任何密码。
2. **保护您的设备安全**：确保您的手机或电脑未感染恶意软件（如键盘记录器）。
3. **定期备份**：由于数据存储在浏览器本地，清除缓存会导致数据丢失，请定期使用 CSV 导出功能进行备份。
